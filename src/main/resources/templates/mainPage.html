Данный Проект является демонстрационным сервисом авторизации.
Регистрация нового пользователя проходит по REST через AJAX.
Авторизированные пользователи видят свой баланс.
Администраторы Сервиса могут дополнительно просматривать список пользователей, а также регистрировать новых пользователей с возможностью задавать им роль админа.

Вход на сервис:
как Admin (логин: Admin пароль: 123);
как User (логин: User, пароль: 123).
